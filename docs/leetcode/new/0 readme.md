- 首先，确定问题场景, 这是对问题空间的描述; 并且问题分类，是哪一类问题。
  > 要跟输入区分开，输入只是为了传递数据
- 然后，具体问题，分析和答案。
- 注意，不要把技巧当作场景、分类。技巧是解空间的描述
  > 两数之和 场景是集合中找成对元素，分类可以以后再分
